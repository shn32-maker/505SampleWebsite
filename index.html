How I provided this custom font

To deliver a custom font into my browser, I used the @font-face method in CSS. Here's how it works:

step1, I picked a unique font called "Righteous" from a CDN (Content Delivery Network). Instead of using Google Fonts, I used jsDelivr which hosts font files.

step2, I added a @font-face rule in my CSS that tells the browser where to download the font file from. The font file is in WOFF2 format, which is compact and works in most modern browsers.

Final step, I applied the font to the body element using the font-family property. Now when I load this page, my browser should download the font file and use it to display all the text!

Good part is that this font most probably is not on my computer, but I can see it anyway because the browser grabbed it from the internet.
